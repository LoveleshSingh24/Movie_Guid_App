# 🎬 Movie Guide App

**Movie Guide** is a sleek and responsive web application that allows users to search for movies and view detailed information like ratings, genre, release date, runtime, cast, plot, and more. Built using HTML, CSS, and JavaScript, the app fetches real-time movie data from the OMDB API.

## 🌟 Features

- 🔍 **Search for Movies** by title.
- 📊 **View Movie Details**: Ratings, genres, cast, plot, runtime, and more.
- 🖼️ **Movie Posters** for visual appeal.
- 💻 Fully responsive design.
- ⚡ **Real-time data** fetched from OMDB API.

## 🚀 Technologies Used

- **HTML5**: For building the structure of the app.
- **CSS3**: To style the UI and make it responsive.
- **JavaScript (ES6+)**: Handles API calls and dynamic content updates.
- **OMDB API**: Provides real-time movie data.

## 🛠️ How It Works

1. Enter a movie title in the search box.
2. The app sends a request to the OMDB API.
3. The app dynamically displays the movie's information, including the poster, rating, genre, plot, and more.
